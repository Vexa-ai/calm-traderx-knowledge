# CalmHub

- **Repo:** finos/architecture-as-code
- **Directory:** calm-hub/
- **Language:** Java (Quarkus)
- **Database:** MongoDB
- **Parent project:** [[architecture-as-code]]

## Overview

CalmHub is the backend REST API for publishing and retrieving CALM architecture documents. Built with Java and the Quarkus framework, it uses MongoDB for storage. It provides endpoints for submitting, querying, and managing architecture models, serving as the central repository for an organization's CALM documents.
