## Overview

The CALM CLI is a command-line tool for working with CALM architecture documents. It provides commands for validating CALM JSON against the specification schema, generating architecture documents from patterns, templating, and producing human-readable documentation from CALM models ("docify").

## Commands

- `validate` — Check a CALM document against the 1.2 schema
- `generate` — Create a CALM document from a pattern
- `template` — Scaffold new CALM projects
- `docify` — Generate human-readable docs from CALM JSON
